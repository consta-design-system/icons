## Иконки

За перечень анимированных иконок отвечает свойство `icons`.

```tsx
const Icon = withAnimateBaseHOC({
  icons: [IconArrowDown, IconArrowUp],
  transition: 300,
  directions: ['down', 'up'],
});
```

## Импорт

```tsx
import { withAnimateBaseHOC } from '@consta/icons/withAnimateBaseHOC';
```

## Свойства

| Свойство           | Тип                              | По умолчанию | Описание                                                                                                             |
| ------------------ | -------------------------------- | ------------ | -------------------------------------------------------------------------------------------------------------------- |
| [`icons`](#иконки) | `IconComponent[]`                | -            | Перечень иконок                                                                                                      |
| `directions?`      | `AnimateIconBasePropDirection[]` | -            | см [AnimateIconBase](##LIBS.LIB.STAND/lib:icons/stand:components-animateiconbase-stable/tab:dev/hash:ориентация)     |
| `transition?`      | `number`                         | `200`        | см [AnimateIconBase](##LIBS.LIB.STAND/lib:icons/stand:components-animateiconbase-stable/tab:dev/hash:время-анимации) |
