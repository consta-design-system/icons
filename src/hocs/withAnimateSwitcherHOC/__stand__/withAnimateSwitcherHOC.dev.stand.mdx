## Иконки

За перечень анимированных иконок отвечает свойство `icons`.

```tsx
const Icon = withAnimateSwitcherHOC({
  icons: [IconArrowDown, IconArrowUp],
  transition: 300,
  directions: ['down', 'up'],
});
```

## Импорт

```tsx
import { withAnimateSwitcherHOC } from '@consta/icons/withAnimateSwitcherHOC';
```

## Свойства

| Свойство                         | Тип                            | По умолчанию | Описание                                                                                                                     |
| -------------------------------- | ------------------------------ | ------------ | ---------------------------------------------------------------------------------------------------------------------------- |
| [`startIcon`](#иконки)           | `IconComponent[]`              | -            | Начальное состояние иконки                                                                                                   |
| [`endIcon`](#иконки)             | `IconComponent[]`              | -            | Конечное состояние иконки                                                                                                    |
| [`startDirection?`](#ориентация) | `AnimateIconBasePropDirection` | `'up'`       | см [AnimateIconSwitcher](##LIBS.LIB.STAND/lib:icons/stand:components-animateiconswitcher-stable/tab:dev/hash:ориентация)     |
| [`endDirection?`](#ориентация)   | `AnimateIconBasePropDirection` | `'up'`       | см [AnimateIconSwitcher](##LIBS.LIB.STAND/lib:icons/stand:components-animateiconswitcher-stable/tab:dev/hash:ориентация)     |
| `transition?`                    | `number`                       | `200`        | см [AnimateIconSwitcher](##LIBS.LIB.STAND/lib:icons/stand:components-animateiconswitcher-stable/tab:dev/hash:время-анимации) |
