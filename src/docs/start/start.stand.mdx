Эта библитека содержит все иконки для дизайн-системы Consta.

## Быстрый старт

#### Установите пакет и зависимости

`yarn add @consta/uikit @consta/icons` или `npm install @consta/uikit @consta/icons`

#### Подключите тему

Тему обязательно нужно подключать в корне приложения, так, чтобы все компоненты `@consta/icons` были вложены в компонент `Theme`.

#### Всë!
