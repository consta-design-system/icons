## Использование

Чтобы использовать анимированные иконки, необходимо обернуть компонент в которых они используются в `AnimateIconSwitcherProvider`.

> Используется в связке с хоком [withAnimateSwitcherHOC](##LIBS.LIB.STAND.TAB/lib:icons/stand:hocs-withanimateswitcherhoc-stable/tab:dev)

```tsx
const Icon = withAnimateSwitcherHOC({
  icons: [IconArrowDown, IconArrowUp],
});

export const AnimateIconSwitcherProvider = () => {
  return (
    <AnimateIconSwitcherProvider active>
      <Icon size="m" view="alert" />
    </AnimateIconSwitcherProvider>
  );
};
```

## Свойства

| Свойство   | Тип               | По умолчанию | Описание              |
| ---------- | ----------------- | ------------ | --------------------- |
| `active?`  | `boolean`         | `false`      | Флаг состояния иконки |
| `children` | `React.ReactNode` | -            | Дочерние компоненты   |
