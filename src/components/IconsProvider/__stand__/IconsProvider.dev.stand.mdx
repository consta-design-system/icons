## Использование

Для использования оптимизированного рендеринга иконок вам необходимо все приложение обернуть в `IconsProvider`.

```tsx
<IconsProvider>
  <IconAdd size="m" view="alert" />
</IconsProvider>
```

## Свойства

| Свойство   | Тип               | По умолчанию | Описание            |
| ---------- | ----------------- | ------------ | ------------------- |
| `children` | `React.ReactNode` | -            | Дочерние компоненты |
