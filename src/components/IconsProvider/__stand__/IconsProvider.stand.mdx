## Использование

Чтобы использовать оптимизированный рендеринг, необходимо обернуть всё приложение в `IconsProvider`.

```tsx
<IconsProvider>
  <IconAdd size="m" view="alert" />
</IconsProvider>
```

## Свойства

| Свойство   | Тип               | По умолчанию | Описание            |
| ---------- | ----------------- | ------------ | ------------------- |
| `children` | `React.ReactNode` | -            | Дочерние компоненты |
