## Иконки

За перечень анимированных иконок отвечает свойство `icons`. Активная иконка определяется через свойство `activeIndex`.

```tsx
export const AnimateIconBaseIcons = () => {
  const [activeIndex, setActiveIndex] = useState(0);

  return (
    <AnimateIconBase
      onClick={() => setActiveIndex((state) => (state + 1 > 2 ? 0 : state + 1))}
      icons={[IconDocAdd, IconClose, IconRoute]}
      activeIndex={activeIndex}
    />
  );
};
```

## Направление

За направление иконки отвечает свойство `directions`, которое может быть равно углу её поворота.

> Если указана одна иконка, то свойство `activeIndex` будет отвечать только за её текущее направление.

```tsx
export const AnimateIconBaseDirections = () => {
  const [activeIndex, setActiveIndex] = useState(0);

  return (
    <AnimateIconBase
      onClick={() => setActiveIndex((state) => (state + 1 > 3 ? 0 : state + 1))}
      icons={[IconDinosaur]}
      directions={[0, 90, 180, 270]}
      activeIndex={activeIndex}
    />
  );
};
```

<AnimateIconBaseDirections />

Одновременное изменение направления и иконки.

```tsx
export const AnimateIconBaseIconsAndDirections = () => {
  const [activeIndex, setActiveIndex] = useState(0);

  return (
    <AnimateIconBase
      onClick={() => setActiveIndex((state) => (state + 1 > 2 ? 0 : state + 1))}
      icons={[IconDocAdd, IconDinosaur, IconClose, IconRoute]}
      directions={[0, 90, 180, 270]}
      activeIndex={activeIndex}
    />
  );
};
```

## Время анимации

Время выполнения анимации в миллисекундах определяется через свойство `transition`. По умолчанию равно 200 мс.

```tsx
<AnimateIconBase
  icons={[IconArrowDown, IconArrowUp, IconArrowLeft]}
  activeIndex={2}
  transition={400}
/>
```

## Свойства

| Свойство                         | Тип                                                                                                           | По умолчанию | Описание                                                                                                  |
| -------------------------------- | ------------------------------------------------------------------------------------------------------------- | ------------ | --------------------------------------------------------------------------------------------------------- |
| [`icons`](#иконки)               | `IconComponent[]`                                                                                             | -            | Перечень иконок                                                                                           |
| [`activeIndex?`](#иконки)        | `number`                                                                                                      | `0`          | Индекс активной иконки                                                                                    |
| [`directions?`](#направление)    | `number[]`                                                                                                    | -            | Перечень направлений для иконок                                                                           |
| [`transition?`](#время-анимации) | `number`                                                                                                      | `200`        | Время анимации в мс                                                                                       |
| `size?`                          | `'xs'`, `'s'`, `'m'`, `'l'`                                                                                   | `'m'`        | Размер (см. [Icon](##LIBS.LIB.STAND.TAB/lib:icons/stand:icons-icon-stable/tab:dev/hash:размер))           |
| `view?`                          | `'alert'`, `'brand'`, `'ghost'`, `'link'`, `'primary'`, `'secondary'`, `'success'`, `'warning'`, `'disabled'` | -            | Внешний вид (см. [Icon](##LIBS.LIB.STAND.TAB/lib:icons/stand:icons-icon-stable/tab:dev/hash:внешний-вид)) |
| `className?`                     | `string`                                                                                                      | -            | Дополнительный CSS-класс                                                                                  |
| `ref?`                           | `React.Ref<HTMLSpanElement>`                                                                                  | -            | Ссылка на корневой DOM-элемент компонента                                                                 |
