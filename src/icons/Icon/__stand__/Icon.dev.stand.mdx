## Внешний вид

За вид иконки отвечает свойство `view`.

```tsx
<IconBag view="alert" />
```

<IconExampleView />

## Размер

За размер иконки отвечает свойство `size`. Варианты: `'xs'`, `'s'`, `'m'`, `'l'`.

```tsx
<IconBag size="xs" />
<IconBag size="s" />
<IconBag size="m" />
<IconBag size="l" />
```

<IconExampleSize />

## Тег для компонента

Свойство `as` отвечает за тег, в который `Icon` оборачивается в HTML-документе.
По умолчанию это `span`. Вы можете поменять этот тег, чтобы добавить компоненту
свойства другого элемента — например, сделайте его ссылкой.

```tsx
<IconLink view="link" as="a" href="https://consta.design" />
```

## Свойства

| Свойство                    | Тип                                                                                                           | По умолчанию | Описание                                                         |
| --------------------------- | ------------------------------------------------------------------------------------------------------------- | ------------ | ---------------------------------------------------------------- |
| [`size?` ](#размер)         | `'xs'`, `'s'`, `'m'`, `'l'`                                                                                   | `'m'`        | Размер                                                           |
| [`view?`](#внешний-вид)     | `'alert'`, `'brand'`, `'ghost'`, `'link'`, `'primary'`, `'secondary'`, `'success'`, `'warning'`, `'disabled'` | -            | Вид иконки                                                       |
| [`as`](#тег-для-компонента) | `keyof JSX.IntrinsicElements`                                                                                 | `'span'`     | HTML-тег, в виде которого этот компонент будет представлен в DOM |
| `className?`                | `string`                                                                                                      | -            | Дополнительный CSS-класс                                         |
| `ref?`                      | `React.Ref<HTMLSpanElement>`                                                                                  | -            | Ссылка на корневой DOM-элемент компонента                        |
