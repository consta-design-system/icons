## Внешний вид

За вид иконки отвечает свойство `view`.

```tsx
<IconBag view="alert" />
```

| `'alert'`                | `'brand'`                | `'ghost'`                | `'link'`                | `'primary'`                | `'secondary'`                | `'success'`                | `'warning'`                | `'disabled'`                |
| ------------------------ | ------------------------ | ------------------------ | ----------------------- | -------------------------- | ---------------------------- | -------------------------- | -------------------------- | --------------------------- |
| <IconExampleViewAlert /> | <IconExampleViewBrand /> | <IconExampleViewGhost /> | <IconExampleViewLink /> | <IconExampleViewPrimary /> | <IconExampleViewSecondary /> | <IconExampleViewSuccess /> | <IconExampleViewWarning /> | <IconExampleViewDisabled /> |

## Размер

За размер иконки отвечает свойство `size`. Варианты: `'xs'`, `'s'`, `'m'`, `'l'`.

```tsx
<IconBag size="xs" />
<IconBag size="s" />
<IconBag size="m" />
<IconBag size="l" />
```

## Свойства

| Свойство                | Тип                                                                                                           | По умолчанию | Описание                                  |
| ----------------------- | ------------------------------------------------------------------------------------------------------------- | ------------ | ----------------------------------------- |
| [`size?` ](#размер)     | `'xs'`, `'s'`, `'m'`, `'l'`                                                                                   | `'m'`        | Размер                                    |
| [`view?`](#внешний-вид) | `'alert'`, `'brand'`, `'ghost'`, `'link'`, `'primary'`, `'secondary'`, `'success'`, `'warning'`, `'disabled'` | -            | Вид иконки                                |
| `className?`            | `string`                                                                                                      | -            | Дополнительный CSS-класс                  |
| `ref?`                  | `React.Ref<HTMLSpanElement>`                                                                                  | -            | Ссылка на корневой DOM-элемент компонента |
